1-1.1js、css的引入方式 

<!DOCTYPE html>
<html>
<head lang="en">
    <meta charset="UTF-8">
    <title></title>
    <style>
        *{
            margin: 0;
            padding: 0;
        }
        div{
            border-radius: 50%;
        }
    </style>
    <!-- 最常用-->
    <!--<link rel="stylesheet" href="css文件的路径"/>-->
    <style>
        @import url(style.css);
    </style>
</head>
<body>
<!--css引入方式-->
<!--
1、行内式:直接写在标签内部 通过style这个标签属性 写入css代码
2、内嵌式:写在style标签里
3、外链式：通过link标签引入css文件
4、导入式：通过@import 引入css 文件

-->
<div style="width: 100px;height: 100px;background: plum" onclick="alert('大家好！')"></div>//行内引入
<!--js的引入方式-->
<!--
1、行内引入：直接写到标签内部  安全性低，几乎不用
2、内嵌式：写到script标签里
3、外链式:通过script标签的src属性  常用
document.write('<script src="js文件的路径"></script>')
-->
<script>
    alert("哈哈")
</script>


相对路径  ../  上一级目录
         ./   当前目录

<script src="./1.js">
    alert('123');//外链式中的script标签中不可以写js代码，写了肯定不会执行
</script>


</body>
</html>
